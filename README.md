 Gym Management System

 Mô tả
Gym Management System là project website quản lý phòng gym được xây dựng bằng PHP và MySQL, phục vụ mục đích học tập và làm dự án đưa vào CV xin thực tập.

Hệ thống hỗ trợ quản lý hội viên, gói tập và thông tin tập luyện một cách đơn giản, dễ sử dụng.

 Công nghệ sử dụng
- PHP
- MySQL
- HTML, CSS
- Git, GitHub

 Chức năng chính
- Quản lý thông tin hội viên
- Quản lý các gói tập gym
- Theo dõi thời hạn gói tập
- Lưu trữ và xử lý dữ liệu bằng MySQL

 Mục tiêu project
- Rèn luyện kỹ năng lập trình web với PHP
- Thực hành làm việc với cơ sở dữ liệu MySQL
- Quản lý mã nguồn và làm việc với Git, GitHub
